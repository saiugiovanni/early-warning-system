Python project for financial market early warning signals, data analysis and visualisation
# Early Warning System for Financial Markets

This repository contains a Python project developed during my MSc in Mathematical Engineering.

The project builds an Early Warning System for financial market stress using macro-financial indicators and machine learning techniques. The notebook includes data loading, preprocessing, stationarity checks, feature scaling, model training, model evaluation and data visualisation.

The models explored include Logistic Regression, Random Forest, Gaussian Copula and an Autoencoder. The project also includes SHAP-based interpretability and a simple toy portfolio backtesting exercise.

The repository is intended as an academic example of Python code used to manage, analyse and visualise data.
